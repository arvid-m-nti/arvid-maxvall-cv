Dokumentation

  • Ett problem jag stötte på var att ibland så fungerar inte svenska bokstäver, alltså Å, Ä och Ö.
    Problemet löste jag genom att googla hur man skrev bokstäverna på HTML.

  • I början så skrev jag styles på varje av mina texter för att göra texten vit.
    Men sedan fattade jag att det var mycket enklare om jag gjorde en class med namnet "white-text" och applicera den till varje text.
    Jag gjorde sedan likadant med den lila ramen, "purple-border", jag använt på alla bilder och med det jag använde för att förminska bilder till rätt storlek, "smaller-image".

  • När jag skulle lägga till bilder så använde jag först bildadressen istället för att ladda ned bilderna.
    Det fungerade, men det var mycket mer komplicerat att använda bildadressen eftersom det var svårare att hålla koll på vilka bilder jag använt.

  • Jag hade använde jag mig av W3Schools och FreeCodeCamp för att till exempel hitta element jag kunde använda och att hitta lösningar till små problem som jag haft.

  • Jag förstod inte om det var mening att man skulle skriva sina styles i CSS-dokumentet eftersom det inte gick att använda styles jag skrivit i "style.css" i mina HTML-dokument.
    Så jag skrev alla mina styles i HTML-dokumenten, och det fungerade bra.
